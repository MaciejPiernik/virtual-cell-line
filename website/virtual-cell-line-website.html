About Virtual Cell Line

Virtual Cell Line is a web application that allows to simulate and explore gene expression dynamics in cellular systems. The simulations are run by a deep learning model trained to predict the expression of masked genes using the expression of unmasked genes.

Key Features

Cell Selection and Filtering

Select cells based on specific characteristics using a powerful query system. Filter cells by inflammation status or gene expression levels to focus on your area of interest.

Gene Perturbation Simulation

Choose a gene to perturb and set target expression level. The app simulates how changes in one gene's expression affect the entire cellular system over time.

Interactive Visualizations

Explore results through intuitive visualizations, including UMAP projections of cell states and dynamic gene expression plots. Easily identify the most impacted genes and track expression changes over time.

Customizable Simulation Parameters

Fine-tune your simulations with advanced options, allowing for precise control over the perturbation process.

How It Works

  1. Select a cell using specific criteria or randomly.
  2. Choose a gene to perturb and set the target expression level.
  3. Run the simulation to see how the perturbation affects other genes.
  4. Analyze results through various visualizations and data tables.
  5. Iterate and explore different scenarios to gain insights into gene regulatory networks.
